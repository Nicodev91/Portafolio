PERRO BOMBA - JUEGO ESTILO BOMBERMAN
====================================

DESCRIPCIÓN
-----------
Perro Bomba es una versión personalizada del clásico juego Bomberman, desarrollada en Python utilizando la biblioteca Pygame. En este juego, controlas a un personaje que debe eliminar a todos los enemigos colocando bombas estratégicamente mientras evitas ser alcanzado por las explosiones o los enemigos.

CÓMO JUGAR
----------
- Usa las teclas de dirección (flechas) para mover a tu personaje por el mapa
- Presiona ESPACIO para colocar una bomba
- Presiona ESC para salir del juego

OBJETIVO
--------
Eliminar a todos los enemigos del nivel sin morir en el intento. Cuando elimines a todos los enemigos, verás el mensaje "Eres una máquina, perrooooo" indicando que has ganado.

ELEMENTOS DEL JUEGO
------------------
- Jugador: Tu personaje, que comienza en la esquina superior izquierda
- Enemigos: Se mueven por el mapa y te harán perder si te tocan
- Bloques indestructibles: Forman el perímetro del mapa y aparecen en patrones dentro del nivel
- Bloques destructibles: Pueden ser destruidos por las bombas
- Bombas: Al explotar, destruyen bloques y enemigos en su radio de acción

CONTROLES
---------
- W-A-S-D: Mover al jugador
- ESPACIO: Colocar bomba
- ESC: Salir del juego

REQUISITOS
----------
- Python 3.x
- Pygame (instalable con: pip install pygame)

EJECUCIÓN
---------
Para iniciar el juego, ejecuta el archivo principal:
python perrobombagame.py

¡Diviértete jugando Perro Bomba!
